# 🏃 Модуль фитнес-трекера (ООП-версия)

Доработанная версия модуля с использованием структур, методов, встраивания и интерфейсов.

## 📦 Состав модуля

| Пакет | Назначение |
|-------|------------|
| `spentenergy` | Расчёт дистанции, скорости и калорий |
| `personaldata` | Структура с данными пользователя |
| `trainings` | Парсинг и вывод информации о тренировках |
| `daysteps` | Парсинг и вывод информации о прогулках |
| `actioninfo` | Интерфейс и функция вывода общей информации |

## 🛠️ Стек

- Golang
- Стандартная библиотека (`time`, `strconv`, `strings`, `log`)

## 🚀 Запуск

```bash
go run main.go
